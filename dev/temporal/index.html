Temporal Metrics

This section provides metrics related to temporal dynamics, such as changes over time or summaries of time-series data.

ADRIAIndicators.absolute_gain! — Method
absolute_gain!(metric::AbstractArray{T,N}, out::AbstractArray{T,M}; time_dim::Int=1)::Nothing where {T,N,M}

Calculate the absolute gain for a metric over time.

The absolute gain is the final value minus the initial value.

Arguments

  • metric : The metric to calculate the gain from, with a time dimension.
  • out : The pre-allocated output array.
  • time_dim : The dimension of the metric array that represents time. Defaults to 1.

ADRIAIndicators.metric_anomalies! — Method
metric_anomalies!(metric::AbstractArray{T,N}, baseline::AbstractArray{T,M}, out::AbstractArray{T,N}; time_dim::Int=1)::Nothing where {T,N,M}

Calculate the standardized anomalies of a metric against a baseline.

The anomaly is calculated as (metric - baseline) / std(baseline). The baseline is expected to have dimensions that can be broadcast to the metric array (e.g., matching all dimensions except the time dimension).

Arguments

  • metric : The metric to calculate the anomaly from.
  • baseline : The baseline to compare against.
  • out : The pre-allocated output array.
  • time_dim : The dimension of the metric array that represents time. Defaults to 1.
source

ADRIAIndicators.relative_gain! — Method
relative_gain!(metric::AbstractArray{T,N}, out::AbstractArray{T,M}; time_dim::Int=1)::Nothing where {T,N,M}

Calculate the relative gain for a metric over time.

The relative gain is the final value minus the initial value, divided by the initial value. Values of NaN or Inf resulting from division by zero are replaced with 0.0.

Arguments

  • metric : The metric to calculate the gain from, with a time dimension.
  • out : The pre-allocated output array.
  • time_dim : The dimension of the metric array that represents time. Defaults to 1.
